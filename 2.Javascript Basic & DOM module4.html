1. What is Javascript ?

-> JavaScript (JS) is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions. While it is most well-known as the scripting language for Web pages

-> Also use it, such as Node.js, Apache CouchDB and Adobe Acrobat. JavaScript is a prototype-based, multi-paradigm, single-threaded, dynamic language, supporting object-oriented, imperative, and declarative (e.g. functional programming) styles.

-> JavaScript's dynamic capabilities include runtime object construction, variable parameter lists, function variables, dynamic script creation , object introspection , and source-code recovery.

8. What is === operator ?

-> === called strict equality operator.

-> checks whether its two operands are equal, returning a Boolean result.

-> Unlike the equality operator, the strict equality operator always considers operands of different types to be different.

Example : console.log(1 === 1); Expected output: true
