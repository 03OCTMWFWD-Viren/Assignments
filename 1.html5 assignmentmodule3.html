1:- What are the new tags added in HTML5 ?

< article >...< /article > tag: The article tag is one of the new sectioning element in HTML5. The HTML article tag is used to represent an article.

< aside >...< /aside > tag: The aside tag is used to describe the main object of the web page in a shorter way like a highlighter.

< audio >...< /audio > tag: The audio tag is used to insert an audio into an HTML webpage.

< canvas >...< /canvas > tag: The canvas tag in HTML is used to draw graphics on a web page using JavaScript.

< datalist >...< /datalist > tag: The datalist tag is used to provide autocomplete feature in the HTML files.

< details >...< /details > tag: The details tag is used for the content/information which is initially hidden but could be displayed if the user wishes to see it.

< embed >...< /embed > tag: The embed tag in HTML is used for embedding external applications which are generally multimedia content like audio or video into an HTML document.

< footer >...< /footer > tag: The footer tag in HTML is used to define a footer of HTML document.

< figure >...< /figure > tag: The figure tag in HTML is used to add self-contained content like illustrations, diagrams, photos, or codes listing in a document.

< header >...< /header > tag: The header tag contains information related to the title and heading of the related content.

< hgroup >...< /hgroup > tag: The hgroup tag in HTML stands for heading group and is used to group the heading elements.

< keygen >...< /keygen > tag: The keygen tag in HTML is used to specify a key-pair generator field in a form.

< mark >...< /mark > tag: The mark tag in HTML is used to define the marked text.

< meter >...< /meter > tag: It is used to define the scale for measurement in a well-defined range and also supports a fractional value.

< nav >...< /nav > tag: The nav tag is used for declaring the navigational section in HTML documents.

< output >...< /output > tag: The output tag in HTML is used to represent the result of a calculation performed by the client-side

< progress >...< /progress > tag: It is used to represent the progress of a task.

< ruby >... /ruby > tag: The ruby tag in HTML is used to specify the ruby annotation which is a small text, attached with the main text

< section >...< /section > tag: The section tag defines the section of documents such as chapters, headers, footers, or any other sections.

< time >...< /time > tag: The time tag is used to display the human-readable date/time.

< wbr >...< /wbr > tag: The wbr tag in HTML stands for word break opportunity and is used to define the position within the text which is treated as a line break by the browser.

< video >...< /video > tag: The video tag is used to embed video content in a document, such as a movie clip or other video streams.

2:- How to embed audio and video in a webpage ?

The HTML5 < audio and video > tags make it simple to add media to a website. You need to set src attribute to identify the media source and include a controls attribute so the user can play and pause the media.

3:-Semantic element in HTML5 ?

HTML was originally created as a markup language to describe documents on the early internet.

Elements such as < header > , < footer > and < article > are all considered semantic because they accurately describe the purpose of the element and the type of content that is inside them.

4:- Canvas and SVG tags ?

The < svg >...< /svg > tag defines a container for SVG graphics. SVG has several methods for drawing paths, boxes, circles, text, and graphic images.

The < canvas >...< /canvas > tag is used to draw graphics, on the fly, via scripting (usually JavaScript). The canvas tag is transparent, and is only a container for graphics, you must use a script to actually draw the graphics.
